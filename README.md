# AMBA-APB-Advance-Peripheral-Bus-
“AMBA-APB is a simple low-bandwidth peripheral bus used for connecting slow peripherals in SoCs. It uses a two-phase protocol (SETUP and ACCESS) with stable address and data phases, ensuring low power, easy timing closure, and efficient read/write transfers.”
